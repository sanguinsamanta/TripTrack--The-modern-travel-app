How it works

Automatic detection + gentle user nudges create accurate trip chains while preserving privacy.

Automatic detection

Sensors (GPS, accelerometer) detect trips and suggest modes (bus, car, walk, cycle).

Quick nudges

Users receive one-tap prompts to confirm purpose and companions.

Privacy-first

Explicit consent, anonymization, and user control before data sharing.

Shareable insights

Aggregated data is shared with NATPAC to improve transport planning.

Key Features

Automatic Trip Detection

Background logging with low-power heuristics and sensor fusion.

Smart Nudges

Contextual one-tap prompts for trip purpose and companions.

Household Mode

Add household members for combined trip chains and group analysis.

Fun Games

Play games and win prizes! Compete with other users, gain more miles and win attractive discounts!
